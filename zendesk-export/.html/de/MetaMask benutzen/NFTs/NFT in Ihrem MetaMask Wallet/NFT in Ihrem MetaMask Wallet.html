Neu bei Krypto und Web3?

Besuchen Sie MetaMask Learn für eine einfache Lernerfahrung, die speziell für Web3-Neulinge entwickelt wurde. Es ist völlig kostenlos, in mehreren Sprachen verfügbar und enthält nützliche Tools wie Simulationen, die Ihnen helfen, sich mit MetaMask zurechtzufinden.

Die Unterstützung für die Anzeige von NFTs in MetaMask variiert je nachdem, wie Sie sie verwenden. NFTs können nicht in der MetaMask-Erweiterung angezeigt werden, sondern können in MetaMask Mobile und in der Portfolio Dapp angezeigt werden.

Wir werden in Kürze die MetaMask Extension mit NFT aktualisieren.

Portfolio Dapp Mobile

Es ist derzeit nicht möglich, NFTs in der MetaMask-Erweiterung anzuzeigen. Stattdessen empfehlen wir Ihnen, die Portfolio Dapp zu besuchen. Sie können entweder direkt über portfolio.metamask.io zur Website gehen oder dem Link „Portfolio“ in MetaMask folgen, direkt unter den Registerkarten „Assets“ und „Aktivität“.

Verwalten Sie Ihre NFTs mit der Portfolio Dapp

Die Portfolio Dapp ist eine eigenständige App, die MetaMask ergänzt und Ihnen hilft, Ihre Token (und NFTs!) zu verwalten. Um Ihre NFTs anzuzeigen, einfach:

  • Starten Sie die Portfolio Dapp
  • Melden Sie sich mit MetaMask bei der Adresse, die Sie sehen möchten
  • Klicken Sie auf die Registerkarte „NFTs“

Alle NFTs, die den Konten, die Sie mit der App verknüpfen, zuzurechnen sind (und die ihnen gehören), sollten automatisch angezeigt werden. Bitte beachten Sie, dass nicht alle NFTs korrekt angezeigt werden können – dies hat damit zu tun, wie wir Daten über sie für Drittquellen abrufen können. Wir arbeiten jedoch an der Maximierung der Kompatibilität, mit dem Ziel, sicherzustellen, dass die überwiegende Mehrheit der NFTs in der Dapp angezeigt wird.

 

NFTs werden unter dem NFT angezeigt. Bitte denken Sie daran, dass einige NFTs in der mobilen App nicht oder nicht korrekt angezeigt werden.

Die einfachste Möglichkeit, Ihre NFTs anzuzeigen, besteht darin, die NFT der App zu aktivieren. Siehe die FAQs für die Vorgehensweise.

Wenn die AutoErkennung Ihre NFTs nicht findet, können Sie sie manuell hinzufügen, indem Sie diese Schritte verwenden:

  1. Finde die Adresse der NFT. Auf OpenSea zum Beispiel auf der NFT's unter Details gibt es einen blauen Hyperlink-Wert mit der Bezeichnung „Contract Address“. Wenn Sie darauf klicken, gelangen Sie zur Adresse des Vertrags auf Etherscan; oben links auf dieser Seite sollte ein Symbol mit der Bezeichnung „Vertrag“ und rechts eine lange Reihe von Buchstaben und Zahlen sein. Dies ist die Adresse des Vertrags, der Ihre NFT erstellt hat. Klicken Sie auf das Symbol „Kopieren“ rechts neben der Adresse, und Sie haben es in Ihrer Zwischenablage.

  2. Tippen Sie in MetaMask Mobile auf die Registerkarte „NFTs“, scrollen Sie nach unten und tippen Sie auf den Link „NFTs importieren“

  3. Fügen Sie die Adresse der NFT aus Ihrer Zwischenablage in das Feld „Adresse“ ein.
  4. Finden Sie die ID des NFT. Die ID eines Collectibles ist eine eindeutige Kennung, da keine zwei NFTs gleich sind. Auch auf OpenSea ist diese Nummer unter „Details“. Notieren Sie sich das oder kopieren Sie es in Ihre Zwischenablage.
  5. Fügen Sie die ID in das Feld mit der Bezeichnung „ID“ in MetaMask Mobile ein, tippen Sie auf die Schaltfläche „HINZUFÜGEN“, und Ihre NFTs sollten unter dem Tab „NFTs“ angezeigt werden.


FAQs

Wie aktiviere ich die NFT Erkennung?

Wenn Sie auf der Startseite der App zum Tab „NFTs“ wechseln, wird eine Eingabeaufforderung angezeigt, wenn Sie die NFT noch nicht aktiviert haben. Folgen Sie dem Link, den es enthält, um das entsprechende Einstellungsmenü aufzurufen, in dem Sie die Autoerkennung aktivieren können.

Bitte beachten Sie: Die automatische NFT ist nur auf dem Ethereum-Mainnet verfügbar. Die Option wird nicht in den Einstellungen angezeigt, wenn Sie ein anderes Netzwerk ausgewählt haben.

Andernfalls folgen Sie diesen Schritten:

  1. Tippen Sie auf das Hamburger-Symbol in der oberen linken Ecke der App auf das Einstellungsmenü.

  2. Finden Sie „Sicherheit & Datenschutz“.

  3. Drücken Sie den Schalter, um die Autodetektion zu aktivieren. Beachten Sie, dass auch „OpenSea API aktivieren“ eingeschaltet sein muss, da sonst der Schalter für die automatische NFT-Erkennung ausgegraut ist.

    Die meisten NFTs sollten jetzt automatisch unter dem Tab „NFTs“ in Ihrem Wallet erscheinen.

     

„Sie sind nicht der Eigentümer dieser NFT“ Fehler

Wenn Sie eine Nachricht erhalten, die Ihnen mitteilt, dass Sie nicht der Eigentümer des NFT sind, stellen Sie bitte sicher, dass Sie versuchen, den Token zum richtigen MetaMask-Konto hinzuzufügen. Überprüfen Sie außerdem im Block Explorer des Netzwerks, dass Sie der NFT-Besitzer Ihrer MetaMask Wallet-Adresse sind. Lesen Sie unseren Leitfaden, wenn Sie sich nicht sicher sind, wie Sie Ihr Wallet in einem Block Explorer überprüfen können.

Wie kann ich überprüfen, ob ich tatsächlich das NFT habe?

Suchen Sie im Block Explorer nach Ihrer MetaMask Wallet-Adresse und überprüfen Sie Ihre Transaktionen, um zu sehen, ob die Übertragung des NFT in Ihr Wallet erfolgreich war. Alternativ können Sie nach der Vertragsadresse des NFT suchen und überprüfen, ob sein Standort mit Ihrer Wallet-Adresse übereinstimmt und ob die Transaktion abgeschlossen ist.

Sie können alle Transaktionen sehen, die mit Ihren NFTs durchgeführt wurden, indem Sie im Block-Explorer auf die Registerkarte „ERC-1155/ERC-721 Token Txns“ klicken. Wenn Sie die NFT hier nicht sehen, kontaktieren Sie den Absender.

„Persönliche Adresse gefunden. Eingabe der Adresse des Abholvertrags.“

Dieser Fehler tritt auf, wenn Sie eine Wallet-Adresse anstelle der Adresse des NFT selbst eingefügt haben. Folgen Sie den Anweisungen unter der Registerkarte „Mobil“ oben, um die Vertragsadresse des NFT zu finden, und fügen Sie diese stattdessen ein.

Wir können nichts tun, wenn die Transaktion nicht in einem Block Explorer vorhanden ist.
