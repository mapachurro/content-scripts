Wenn Sie Gelder von Ihrem Binance an MetaMask gesendet haben und sie nicht in Ihrem Wallet sehen können, empfehlen wir Ihnen, diese Schritte auszuführen:

Schritt 1: Überprüfen Sie die Transaktion in einem Blockchain-Explorer

Zuerst müssen Sie überprüfen, welches Netzwerk Binance zum Abschluss der Transaktion verwendet hat. Sie können das tun, indem Sie die Rückzugsdaten auf Binance überprüfen:

Dann gehen Sie zum Netzwerkexplorer. Hier sind einige Beispiele:

  • Etherscan für das Ethereum-Mainnet
  • Bscscan für BSC (Binance Smart Chain)
  • Polygonscan für Polygon
  • Snowtrace für Avalanche

Sobald Sie sich im Blockchain-Explorer befinden, suchen Sie entweder nach Ihrer MetaMask-Adresse oder nach der Transaktions-ID, um die Details der Transaktion aufzurufen und zu bestätigen, dass sie abgeschlossen wurde.

Im Übersichtsabschnitt auf der Seite Ihres Kontos im Block-Explorer können Sie Ihre Token-Salden sehen. Wenn es sich bei dem Token um den nativen Token (BNB, ETH, MATIC, AVAX) handelt, wird er im Abschnitt „Balance“ angezeigt (wie im Screenshot oben).

Sofern es sich nicht um einen nativen Token handelt, wird dieser im Dropdown-Menü im Abschnitt Token angezeigt.

 

Schritt 2: Fügen Sie das Netzwerk zu MetaMask hinzu und zeigen Sie den Token bei Bedarf an

Wenn Sie ein Netzwerk hinzufügen möchten, lesen Sie bitte unsere Anleitung zum Hinzufügen mit Chainlist, oder unseren Artikel, der sich mit dem Hinzufügen eines benutzerdefinierten Netzwerks beschäftigt.

Wenn der Token nach dem Hinzufügen des Netzwerks nicht der native Token dieses Netzwerks ist, müssen Sie ihn als benutzerdefinierten Token anzeigen. 

Gehen Sie im Block-Explorer zum Abschnitt Token und klicken Sie auf den Token, den Sie erhalten haben:

 

Um den benutzerdefinierten Token in MetaMask aufzunehmen, müssen Sie Folgendes tun:

  1. Die Vertragsadresse des Tokens kopieren. Eine Anleitung finden Sie hier; grundsätzlich müssen Sie den Token im Block-Explorer finden und dort seine Adresse abrufen.
  2. Gehen Sie zu MetaMask und suchen Sie unter der Registerkarte „Aktiva” nach „Token importieren”. Wenn Sie bereits viele Token hinzugefügt haben, müssen Sie möglicherweise ein wenig scrollen, um dies zu finden. Klicken Sie darauf, um den Token hinzuzufügen und geben seine Adresse ein.

Wenn Sie die Token im falschen Netzwerk verschickt haben, können Sie jederzeit eine Brücke verwenden, um den betreffenden Token in das gewünschte Netzwerk zu übertragen.

Fürs Bridging von Token zwischen den gängigsten Netzwerken können Sie Multichain (ehemals AnySwap) verwenden.

Fürs Bridging der Token müssen Sie über Guthaben in den Token des Netzwerks verfügen, damit Sie die Gasgebühren bezahlen können. Das ist bei jeder Transaktion notwendig. Für BSC benötigen Sie zum Beispiel BNB, für Polygon MATIC und für Avalanche AVAX.
